# Phishing with Lateral Movement

## Incident Overview
A phishing attack resulting in credential compromise and lateral movement within the enterprise environment.

## Initial Detection
- Indicators of suspicious email activity
- Abnormal authentication attempts
- Unusual endpoint or network behavior

## Investigation & Analysis
- Email analysis
- Endpoint behavior review
- Authentication and access logs
- Network traffic analysis

## Containment Actions
- Isolate affected endpoints
- Disable compromised accounts
- Block malicious indicators

## Eradication Steps
- Remove malicious artifacts
- Reset credentials
- Apply relevant security patches

## Recovery
- Restore systems to normal operation
- Monitor for recurring activity
- Validate security controls

## Lessons Learned
- Gaps identified during the incident
- Recommended improvements
